# 💰 Controle Financeiro


Sistema de gerenciamento financeiro desenvolvido em **PHP, JavaScript e MySQL** durante meus estudos em Engenharia de Software.


## Tecnologias


- PHP
- JavaScript
- HTML5
- CSS3
- MySQL


## Funcionalidades


- Login de usuários
- Dashboard financeiro
- Cadastro de receitas e despesas
- Histórico de transações
- Integração com banco de dados MySQL

## Sobre o projeto


Este projeto foi desenvolvido para praticar conceitos de desenvolvimento Full Stack, incluindo autenticação de usuários, operações CRUD, integração com banco de dados e construção de uma interface intuitiva para controle financeiro.
